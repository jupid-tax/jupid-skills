# IRS Transcript Types

The IRS offers five transcript types (and two additional rare ones). Picking the right one matters: requesting the wrong type wastes 5-10 days and may force a re-submission.

This file explains each type, what it shows, what it doesn't, when to use it, and how it differs from related types.

---

## 1. Tax Return Transcript (Form 4506-T Line 6a)

### What it shows

Most line items from the originally-filed Form 1040 series:
- Filing status (Single, MFJ, MFS, HoH, QW)
- Adjusted Gross Income (AGI) — line 11 of Form 1040 for 2024
- Taxable income — line 15
- Total tax — line 24
- Federal income tax withheld — line 25a
- Earned Income Credit — line 27
- Total payments — line 33
- Refund or balance due
- Schedules attached: Schedule A (itemized), Schedule B (interest/dividends), Schedule C (self-employment), Schedule D (capital gains), Schedule E (rental/royalty), Schedule SE (self-employment tax), etc., in summary form
- Information from Schedule 1 (additional income/adjustments), Schedule 2 (additional taxes), Schedule 3 (additional credits)

### What it doesn't show

- Changes after original filing — amendments (Form 1040-X), IRS adjustments via correspondence audit (CP2000), examination changes
- Detailed line-by-line breakouts of attached schedules
- Penalty / interest charges
- Payment history (estimated payments, post-return payments)
- The user's actual signature or W-2 attachments

### When to use

- **Mortgage applications** — Lenders almost always specifically request "Tax Return Transcript" for the last 1-2 years to verify income claims
- **Student loan applications** — College financial aid offices, especially private student loans
- **Business loan / SBA applications** — Banks requesting income verification for self-employed borrowers
- **Apartment rental applications** — Some landlords for high-end rentals
- **Personal records** — Quick reference for "what did I make / pay last year"

### Availability

Current tax year + 3 prior tax years.

For older years, use Tax Account Transcript (broader date range) or order a full Form 4506 copy of return ($43, 60-75 days).

### Comparison to actual return

Tax Return Transcript is a "summary view" of what the user filed. It's not the actual signed return, and it doesn't include every supporting schedule's full detail. For most income-verification purposes, that's enough. For tax preparation re-filing or amending, you may want a full Form 4506 copy.

## 2. Tax Account Transcript (Form 4506-T Line 6b)

### What it shows

Activity on the user's IRS tax account for the year requested:
- Filing date
- Return type filed
- Marital status / filing status
- AGI
- Taxable income
- Total tax assessed
- Payments and credits
- Refunds issued (with dates and methods)
- Penalties and interest assessed
- IRS adjustments (CP2000 changes, examination results)
- Notices sent (with dates)
- Account balance
- Codes for various transactions (e.g., code 150 = original return filed; code 766 = credit posted; code 826 = refund offset)

### What it doesn't show

- The income line items as filed (use Return Transcript for that)
- Wages by employer (use Wage and Income Transcript)
- The actual return form

### When to use

- **Disputing IRS balance** — When IRS sends a notice claiming the user owes money, the Account Transcript shows what the IRS has assessed and when
- **Tracking refund delays** — Code 846 shows refund issuance date
- **Verifying payment history** — Estimated tax payments, payments with return, post-return payments
- **Audit defense** — Showing the timeline of IRS actions on the account
- **Confirming amendment processing** — After filing 1040-X, the Account Transcript shows the amended assessment

### Availability

Many years back — often 10+ years for individual accounts. Older history may be archived; call IRS for very old years.

### Reading the codes

The Account Transcript uses three-digit transaction codes (TC). Common ones:
- TC 150 — Tax return filed
- TC 290 — Additional tax assessed (audit, IRS adjustment)
- TC 706 — Generated overpayment applied from another tax module
- TC 766 — Credit to your account
- TC 768 — Earned Income Credit
- TC 806 — W-2 or 1099 withholding
- TC 826 — Credit transferred out
- TC 846 — Refund issued
- TC 971 — Notice issued
- TC 977 — Amended/duplicate return processed

The IRS's own "Transaction Codes Pocket Guide" is the canonical reference. For high-stakes interpretation, consult a CPA or enrolled agent.

## 3. Record of Account Transcript (Form 4506-T Line 6c)

### What it shows

Combination of Tax Return Transcript (6a) + Tax Account Transcript (6b) for the same year.

### When to use

- **Responding to a CP2000 notice** — You need both the original return data (to show what you reported) and the account activity (to show what the IRS adjusted)
- **Comprehensive audit response** — One document covers both filing data and account history
- **Most thorough single-document option** — When you're not sure which type you need, and you need depth

### Availability

Current tax year + 3 prior tax years. (Same as Return Transcript.)

For older years, request Account Transcript separately (since it's available for longer).

## 4. Verification of Non-filing Letter (Form 4506-T Line 7)

### What it shows

A formal letter from the IRS stating they have **no record of a Form 1040** (or other specified form) filed for the requested year(s).

### What it doesn't show

It doesn't certify the user was not required to file. It only certifies the IRS hasn't received a return. If the user filed but the return was lost in the mail, the letter will say "no record" — which may misrepresent the user's compliance.

### When to use

- **FAFSA** — Federal student aid forms require parents and students who didn't file taxes to provide a Verification of Non-filing Letter (older years; recent FAFSAs use IRS Data Retrieval Tool which checks automatically). The school's financial aid office specifies which year(s).
- **Immigration applications** — Some visa, green card, and citizenship applications require proof of non-filing for years the applicant was not present in the US (or had no US income)
- **Court proceedings** — Divorce, child support, bankruptcy may require this for years where one party didn't file
- **Public benefits applications** — Some state aid programs require non-filing letters for low-income years

### Availability

Any year where the user did not file. Available year-round.

### When NOT to use

If the user did file but the return was rejected, lost, or amended, this letter is misleading. They should:
- Tax Return Transcript or Tax Account Transcript for years they did file
- Verification of Non-filing only for years they truly didn't file

## 5. Wage and Income Transcript (Form 4506-T Line 8)

### What it shows

Third-party-reported income data for the year requested:

- **W-2** — Employer wage reports (showing wages, federal withholding, Social Security, Medicare wages, employer)
- **1099-NEC** — Nonemployee compensation (contractor income)
- **1099-MISC** — Miscellaneous income (royalties, rents, prizes, attorney fees)
- **1099-K** — Third-party network payments (PayPal, Venmo, Stripe, marketplace sales)
- **1099-DIV** — Dividend income
- **1099-INT** — Interest income
- **1099-R** — Retirement / pension distributions (IRA, 401(k), pension)
- **1099-B** — Broker transactions (sales of stocks, bonds)
- **1099-G** — Government payments (unemployment, state tax refunds)
- **1099-S** — Real estate transactions (typically home sales)
- **1099-OID, 1099-Q, 1099-SA** — Less common
- **1098** — Mortgage interest (filed by lender showing interest paid by borrower)
- **1098-T** — Tuition statements (filed by college)
- **1098-E** — Student loan interest (filed by lender)
- **K-1 (partial)** — Partnership / S-corp distributive share (some K-1 data appears, not all)
- **Form 5498** — IRA contributions, fair market value
- **W-2G** — Gambling winnings

### What it doesn't show

- The user's actual return (use Return Transcript for that)
- Self-employment income that was NOT reported by a third party (cash income, foreign income, etc.) — only what was reported TO the IRS by an issuer
- Account transactions / payments (use Account Transcript)

### When to use

- **Replacing lost W-2 / 1099 documents** — Most common use. If the user lost their W-2 or 1099-NEC, the Wage and Income Transcript shows what was reported to the IRS, which the user can then use to prepare or reconstruct a return.
- **Identity theft cases** — Showing the user what income was reported in their name (e.g., a fraudster filed a return using their SSN; the W&I transcript shows what employers/issuers actually reported, so the user can identify forged 1099s).
- **Reconstructing prior-year financials** — When the user is reconstructing several years of tax returns (e.g., after long non-filing).
- **Audit defense** — Confirming the income reported to IRS matches what the user reported on the return.

### Availability

**Last 10 years**. The IRS retains W&I data for 10 years.

### Critical timing caveat

Third parties (employers, banks, brokers, etc.) report their forms to the IRS by **January 31** of the year following the tax year. The IRS processes and indexes these reports throughout spring and summer.

For the **current** tax year (e.g., requesting 2025 W&I in early 2026), the data may not be available until **mid-summer** (June-July) of 2026 at earliest. If the user requests too early, the W&I transcript will be empty or incomplete.

For prior years (1+ years back), the data is fully available year-round.

## Comparison table

| Need | Best transcript |
|------|-----------------|
| Mortgage / lender wants to verify last 2 years' income | Tax Return Transcript |
| FAFSA: didn't file taxes | Verification of Non-filing |
| FAFSA: did file taxes | (use FAFSA's IRS Data Retrieval Tool, fallback to Tax Return Transcript) |
| Lost W-2 / 1099, need to file | Wage and Income Transcript |
| Disputing IRS notice / balance | Tax Account Transcript or Record of Account |
| Replying to CP2000 | Record of Account |
| Identity theft on tax return | Wage and Income Transcript + Account Transcript |
| Confirming amendment processed | Account Transcript |
| Tracking refund delay | Account Transcript (look for TC 846) |
| Years 4+ ago | Account Transcript (more years available) |
| Years 1-3 ago | Any of: Return, Account, Record of Account |
| Years 1-10 ago, third-party-reported income | Wage and Income |
| Need full COPY of original return | NOT 4506-T — use Form 4506 ($43, 60-75 days) |

## Common confusion points

### "I need my W-2 from a previous job"

Best path:
1. Contact the former employer first — they should provide W-2 reissue
2. If unreachable, IRS Wage and Income Transcript (Line 8 of 4506-T)

The W&I Transcript shows the W-2 data the employer reported to IRS, which is enough to file a return. It is NOT the same as a reissued W-2 from the employer.

### "I need my prior return for tax preparation"

If preparing a current return and just need numbers from last year:
- Tax Return Transcript is enough (shows AGI, refund amount, etc.)

If preparing an amended return or need to see specific schedules in detail:
- Form 4506 full COPY is better ($43, 60-75 days)

### "FAFSA wants my parent's tax info"

- If parent filed: tell parent to use FAFSA's IRS Data Retrieval Tool (built into FAFSA application form). No 4506-T needed.
- If parent didn't file: parent needs Verification of Non-filing Letter (Line 7 of 4506-T)

### "I'm a small business owner; I need my business tax transcript"

- Sole proprietor with Schedule C: that's part of the personal 1040; request Tax Return Transcript for "1040"
- LLC taxed as partnership: request "1065" transcript using the LLC's EIN
- LLC taxed as S-corp: request "1120-S" transcript using the LLC's EIN
- C-corporation: request "1120" transcript using the corporation's EIN
- Single-member LLC (disregarded entity): falls back to the owner's 1040 transcript

For business returns, the form is signed by an authorized officer (CEO, President, partner) with the title in the signature block.
