# Form 4506-T Common Mistakes

Form 4506-T looks simple — one page, a few boxes. But the IRS rejects a meaningful percentage of submissions for predictable reasons. Here are the 10 most common errors and how to avoid them.

---

## 1. Multiple transcript types selected

**Mistake**: User checks both 6a (Return Transcript) and 6b (Account Transcript) on the same form, hoping to get both.

**Why it's wrong**: Form 4506-T accepts exactly **one** transcript type per form. The IRS rejects forms with multiple types selected.

**Fix**: File **two separate** Form 4506-T forms — one for each type. Both can be mailed/faxed in the same envelope/transmission to save effort, but each is its own form.

If the user needs multiple types for the same year, also consider:
- 6c (Record of Account) — combines 6a + 6b in one document. Saves a form.
- Get Transcript Online — pull all five types as needed, free, instant.

---

## 2. Signature older than 120 days

**Mistake**: User signs the form, sets it aside, mails it 6 months later when the lender finally asks. IRS rejects with "signature too old."

**Why it's wrong**: Per current Form 4506-T instructions, the taxpayer's signature is valid for **120 days** from signing. If the IRS receives a form signed > 120 days ago, it's rejected.

**Fix**: Sign close to the submission date. If the user is preparing the form in advance, leave the signature field blank and have them sign right before mailing/faxing.

---

## 3. Wrong address on Line 4

**Mistake**: User filled Line 4 with the address they lived at "back when [some event]" rather than the address on the most recent tax return filed.

**Why it's wrong**: Line 4 is for **previous address shown on the last return filed**, not arbitrary prior addresses. The IRS uses Line 4 to verify identity by matching against their records.

If the user has not moved since their last return, Line 4 should be **blank** (or "Same as Line 3").

If the user has moved since the last return, Line 4 = address on most recent return; Line 3 = current address.

**Fix**: Confirm with the user the address on their most recent return. If unsure, look it up: Get Transcript by Mail uses this address for identity check; if it works there, it's the right address for Line 4.

---

## 4. Joint return missing spouse signature

**Mistake**: One spouse signs Form 4506-T to request a joint return transcript; the other spouse doesn't sign.

**Why it's wrong**: For joint returns, the IRS considers both spouses to be taxpayers. To release a joint transcript, **both spouses must sign**. A single-spouse-signed form for a joint transcript is rejected.

**Fix**: Either:
- Have both spouses sign the same form (best)
- Or, have only the requesting spouse file the form, but check Line 6a / 6b for *their own* portion of the return data (not always practical for joint returns)
- Or, the absent spouse can authorize via Form 2848

---

## 5. Form filled for "current year" but return not yet processed

**Mistake**: User filed their 2024 return in late February, then immediately submitted Form 4506-T for tax year 2024 in early March. IRS rejects.

**Why it's wrong**: The IRS doesn't have processed transcript data until ~2-3 weeks after e-file acceptance, and 6-8 weeks after paper filing.

**Fix**: Wait for the return to be processed before requesting a transcript. The agent can check by:
- Looking up the return status via Where's My Refund (https://www.irs.gov/refunds) — if status is "Refund issued" or "Return received" with a checkmark, the return is processed
- Or wait 3-4 weeks past filing date for e-file, 8-10 weeks for paper

If the user needs the transcript urgently for a deadline-driven application (mortgage closing, etc.) and the return isn't yet processed, alternatives:
- Tax Account Transcript may be available faster than Return Transcript
- Provide the lender with the actual filed return + acknowledgment of e-file acceptance instead of waiting

---

## 6. Wrong year format

**Mistake**: User enters "2024" on Line 9 instead of "12/31/2024".

**Why it's wrong**: Form 4506-T expects MM/DD/YYYY format for tax years. Calendar-year filers use 12/31/YYYY (the fiscal year-end date). Some IRS clerks may accept "2024" but others reject.

**Fix**: Format as 12/31/YYYY for calendar-year filers. For fiscal-year filers, use the actual fiscal year-end (e.g., 06/30/2024).

---

## 7. Asking for a transcript for a year not yet ended

**Mistake**: In November 2025, user enters 12/31/2025 on Line 9, expecting to get a "current-year-in-progress" transcript.

**Why it's wrong**: The 2025 tax year doesn't conclude until 12/31/2025. There is no return to transcript before the tax year ends and the user files. The earliest a 2025 transcript becomes available is ~2-3 weeks after the user files in early 2026.

**Fix**: Wait until the user has filed and the return has been processed. For mid-year financial verification, the user can provide:
- Year-to-date pay stubs from employer
- Most recent W-2 (last full year)
- Bank statements for income evidence

---

## 8. Wage and Income Transcript requested too early

**Mistake**: In February 2026, user requests the 2025 Wage and Income Transcript expecting to see all their W-2s and 1099s.

**Why it's wrong**: Third parties (employers, banks, brokers, etc.) report their 1099s and W-2s to the IRS by January 31 of the following year. The IRS then processes and indexes these reports throughout spring and summer. Wage and Income Transcripts for the most recent year may not be complete until **June-July** of the following year.

**Fix**: For the most current year's W&I data, wait until at least mid-summer. For prior years, data is fully available year-round.

If the user needs W-2 data urgently and their employer's W-2 wasn't received:
- Contact employer to reissue
- Last resort: Form 4852 (Substitute for W-2) — file return with estimated W-2 data and an explanation

---

## 9. Sending to wrong RAIVS office

**Mistake**: User mails Form 4506-T to the IRS office address from a 2018 instruction page (or a Google search result for "IRS Form 4506-T address").

**Why it's wrong**: IRS RAIVS office addresses change over time. Old addresses may forward, but often delays processing. Worse, the wrong office may simply return the form.

**Fix**: Use the routing chart on **page 2 of the current Form 4506-T** (downloaded fresh from https://www.irs.gov/pub/irs-pdf/f4506t.pdf). The chart maps the user's state of residence (Line 3) to one of three RAIVS offices (Kansas City, Ogden, or Austin) and provides current addresses + fax numbers.

If unsure, use fax instead of mail — fax numbers are also state-specific but are more error-tolerant (if wrong, fax bounces immediately).

---

## 10. Filling Line 5 with wrong third-party address

**Mistake**: User fills Line 5 with what they think is their lender's mailing address, but it's actually the lender's branch office address (not the loan-processing center). Transcript goes to the branch, gets lost or delayed.

**Why it's wrong**: Lenders often have multiple addresses — branch, processing center, underwriting, IVES department. The wrong address means the transcript goes to a place that doesn't know what to do with it.

**Fix**: 
- Ask the lender's loan officer **explicitly**: "What's the exact address for IRS to send my Form 4506-T transcript?"
- The lender often has a customer file number or loan number to include — ask for that too
- Verify by reading back the address before submitting

If the user is unsure, leave Line 5 blank and have IRS mail to the user's home address (Line 3). The user can then forward the transcript to the lender.

---

## Bonus: 11. Confusing 4506 vs. 4506-T vs. 4506-C

This one's so common it deserves an honorable mention.

| Form | What it gets | Cost | Turnaround |
|------|--------------|------|-----------|
| 4506 | Full COPY of return with attachments | $43 per return | 60-75 days |
| 4506-T | Transcript (summary) of return | FREE | 5-14 days |
| 4506-C | IVES transcript (lender-driven) | $2 (lender pays) | 1-2 days |

If the user has been told to fill out "Form 4506" but the lender just wants income verification, the **transcript** (4506-T) is enough — save the user $43.

If the user has Form 4506-C from a lender, sign that and give it back. Don't also fill out 4506-T.

---

## Validation the agent should run

Before submitting Form 4506-T:

- [ ] Exactly one transcript type checked (6a, 6b, 6c, 7, or 8)
- [ ] Year(s) requested are formatted MM/DD/YYYY
- [ ] Year(s) requested are tax years that have ended AND the return has been processed (or, for non-filing letter, year is past)
- [ ] Line 1a name matches return; Line 1b SSN/ID is correct
- [ ] If joint, Line 2a/2b filled and both spouses signing
- [ ] Line 3 = current address; Line 4 = prior return address (if user moved)
- [ ] Line 5 third-party address verified with lender (if used)
- [ ] Form number entered (1040, 1040-SR, etc.)
- [ ] Signature within 120 days of expected IRS receipt
- [ ] Routing address / fax number matches user's state per current IRS chart
- [ ] If using a lender's 4506-C, Form 4506-T is NOT also being submitted (avoid duplicate)

If any check fails, **stop and surface** to the user. Don't submit a form likely to be rejected — it wastes 10-14 days.
