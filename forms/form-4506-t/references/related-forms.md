# Form 4506-T vs. Related Forms

Several IRS forms have similar names and overlapping purposes. Picking the wrong one wastes weeks and may cost the user money. This file documents each form's purpose and when to use which.

---

## Form 4506-T — Request for Transcript of Tax Return (this skill)

- **What**: Request a transcript (summary) of a tax return, account activity, wage/income data, or non-filing letter
- **Cost**: FREE
- **Turnaround**: 5-14 days
- **Who signs**: the taxpayer themselves
- **What you get**: an IRS-generated summary document, not the actual return

Use Form 4506-T for most income-verification and account-history needs.

---

## Form 4506 — Request for Copy of Tax Return

- **What**: Request a **full copy** of an originally-filed Form 1040 (or other) with all attachments — W-2s, 1099s, schedules, signed pages
- **Cost**: $43 per return (per IRS, current fee — verify before filing)
- **Turnaround**: 60-75 days (per IRS, often longer)
- **Who signs**: the taxpayer themselves
- **What you get**: a photocopy of the actual signed return as filed

### When to use Form 4506 instead of 4506-T

- The user needs to **file an amended return** for a year far in the past and doesn't have their own copy of the original
- A court proceeding (divorce, IRS Tax Court) requires the **actual** signed return as evidence
- The user is reconstructing tax history and a transcript summary won't suffice
- The user needs to see attached W-2s / 1099s in the actual return form (NOTE: a Wage and Income Transcript shows the SAME data, faster and free — try that first)
- An auditor or IRS Appeals officer specifically requested the full return

### When Form 4506-T is enough (don't pay $43)

- Lender wants income verification → Tax Return Transcript (free)
- Replacing lost W-2 → Wage and Income Transcript (free)
- FAFSA → Tax Return Transcript or Verification of Non-filing (free)
- Disputing IRS balance → Tax Account Transcript (free)
- Personal records → any of the five free transcript types

## Form 4506-C — IVES Request for Transcript of Tax Return

- **What**: Same purpose as Form 4506-T (transcript request) but routed through IRS's **Income Verification Express Service (IVES)**
- **Cost**: $2 per transcript (paid by the lender, not the user)
- **Turnaround**: 1-2 business days (much faster than 4506-T)
- **Who signs**: the taxpayer
- **Who submits**: the IVES participant (lender, accountant, IRS-authorized third party)

### When Form 4506-C is used

- The user is applying for a mortgage with a major lender (Quicken, Wells Fargo, Chase, Rocket Mortgage, SoFi, etc.)
- The lender is an IRS-approved IVES participant
- The lender provides the user a pre-filled Form 4506-C to sign

### What the agent should do

If the user has Form 4506-C from a lender:
- **Sign and return it to the lender** — do NOT also submit Form 4506-T
- The lender submits via IVES; the user gets the transcript in their loan file in 1-2 days
- The user does not pay the $2 fee; the lender pays it

If the user has both 4506-T and 4506-C from confused parties, use the 4506-C if a lender provided it. Form 4506-T is the fallback.

### Why this exists

IVES was created to give high-volume lenders a faster, dedicated channel. Without IVES, lenders requesting hundreds of transcripts a day would overwhelm RAIVS.

## Form 2848 — Power of Attorney and Declaration of Representative

- **What**: Authorizes a third party (CPA, enrolled agent, attorney, family member) to **represent** the taxpayer before the IRS — file forms, negotiate with IRS, attend audits, sign certain documents
- **Cost**: FREE
- **Turnaround**: 4-6 weeks for IRS to process; representative can act immediately upon receipt
- **Who signs**: the taxpayer + the representative
- **Power granted**: full representation, including signing documents and discussing the case with IRS

### When to use Form 2848 instead of 4506-T

- The user wants their **CPA / EA / attorney** to handle ALL tax matters going forward, not just request transcripts
- The user is in an **audit** and wants their representative to engage with IRS directly
- Ongoing tax disputes, collection issues, or examinations

### Form 2848 vs. 4506-T

| Need | Use |
|------|-----|
| One-time transcript request, deliver to lender | Form 4506-T (Line 5 to lender) |
| Authorize CPA to handle audit on user's behalf | Form 2848 |
| CPA needs to access transcripts repeatedly for ongoing work | Form 8821 (read-only) or Form 2848 (full POA) |

---

## Form 8821 — Tax Information Authorization

- **What**: Authorizes a third party to **receive** the user's tax information (transcripts, notices, account info) — but NOT to represent or sign on behalf of the user
- **Cost**: FREE
- **Turnaround**: 4-6 weeks for IRS to process
- **Who signs**: the taxpayer
- **Power granted**: read-only access to specified tax periods and types

### When to use Form 8821 instead of 4506-T

- The user wants their **accountant / CPA** to be able to pull transcripts on demand for ongoing work — not just one transcript
- Ongoing financial planning where the user wants advisors to have continuous read-only access
- Receiving copies of all IRS notices addressed to the user

### Form 8821 vs. 4506-T

- 4506-T = ONE transcript, ONE time, mailed to user or one third party (Line 5)
- 8821 = ONGOING read-only access to designated information for designated periods

For repeated access, file 8821 once. For a single request, 4506-T is simpler.

---

## Form 8822 — Change of Address

- **What**: Notifies IRS of address change for individual filers
- **Cost**: FREE
- **Turnaround**: 4-6 weeks for IRS to update records

### Relevance to 4506-T

If the user moved and Get Transcript by Mail / IRS notices go to old address, Form 8822 must be filed before transcript requests will be successful.

For Form 4506-T paper requests, the agent can still get a transcript by entering both Line 3 (current address) and Line 4 (prior return address) — the IRS uses Line 4 to verify identity, not for delivery. So 4506-T works without filing 8822 first.

For Get Transcript by Mail, the IRS uses the address on file and does not honor a different address provided in the request — Form 8822 is needed.

For ID.me online auth, the IRS-on-file address and the user's current address must match. If they don't, ID.me may fail; file Form 8822 to fix.

## Form W-2 / W-2 Reissue

If the user lost a W-2, the FIRST step is asking the employer to reissue it — not requesting a transcript.

- The employer is legally required to provide a W-2 to each employee by January 31
- If lost, employers typically can reissue from their payroll system
- Reissue is free and faster than a transcript

If the employer is unreachable, out of business, or refuses, then:
- Wage and Income Transcript (Form 4506-T Line 8) is the IRS fallback
- Form 4852 (Substitute for W-2) is used to file a return without a W-2 if both employer and IRS data are unavailable

## Quick decision matrix

| User's actual need | Right form |
|--------------------|-----------|
| Mortgage application income verification | 4506-T (free) — or use lender's 4506-C if provided |
| Lost W-2 | Ask employer first; if no luck, 4506-T Line 8 (Wage and Income) |
| FAFSA: parent didn't file | 4506-T Line 7 (Verification of Non-filing) |
| FAFSA: parent did file | Use FAFSA's IRS Data Retrieval Tool (no form needed) |
| IRS sent CP2000; need to respond | 4506-T Line 6c (Record of Account) |
| Ongoing CPA work / audits | 8821 (read-only) or 2848 (representation) |
| Court evidence: actual signed return | 4506 ($43, 60-75 days) |
| Lender uses IVES | 4506-C (lender provides) |
| User moved and IRS has wrong address | 8822 first, then 4506-T or Get Transcript |
