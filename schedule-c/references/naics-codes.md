# NAICS Codes for Schedule C Line B

The IRS uses 6-digit NAICS codes (North American Industry Classification System) on Schedule C Line B. Below are codes commonly chosen by solo filers, freelancers, and single-member LLCs. The official IRS list is in the Schedule C instructions, and the Census Bureau maintains the canonical taxonomy at https://www.census.gov/naics/.

The choice matters for two reasons:
1. **IRS audit benchmarks**. The IRS compares expense ratios to peer averages by NAICS code. Choosing a code where 70% expense ratio is normal makes a 70% expense ratio unremarkable; choosing one where 30% is normal makes the same return look like an outlier.
2. **Manufacturing vs. service**. Manufacturing codes (339xxx, 311xxx, etc.) come with an expectation of inventory and COGS. If the user makes physical products, prefer a manufacturing code over a generic retail one.

## Picking a code when nothing fits

The catch-all codes are:

- `999999` — Unable to classify (rarely a good idea; raises eyebrows)
- `541990` — All other professional, scientific, and technical services
- `561990` — All other support services
- `812990` — All other personal services

Use these only if no specific code from the IRS table matches. The IRS instructions list ~470 codes; almost every business fits one of them.

## Changing the code year-over-year

A change in the NAICS code on Schedule C from one year to the next implies a business change. If the actual business hasn't changed, keep the same code — the IRS may flag a code change for review. If the business genuinely changed (pivoted from photography to video editing, say), update the code and be ready to explain.
